The following events are monitored in this specification:

1) Event name: charCount
   Kind: event
   Timing: before
   Parameters: int codePoint
   Pointcut: call(* Character.charCount(int)) && args(codePoint)

2) Event name: toCodePoint
   Kind: event
   Timing: before

   Pointcut: call(* Character.toCodePoint(char, char)) && args(high, low)

The violation message that must be used in the JSON output is:
