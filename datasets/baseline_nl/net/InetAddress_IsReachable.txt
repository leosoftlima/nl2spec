This specification is written in the EVENT formalism.

This specification is called InetAddress_IsReachable.


The following events are monitored in this specification:

isreachable is an event observed before the call. It takes int timeout as parameter. It matches: call(* InetAddress+.isReachable(int)) && args(timeout).

isreachable is an event observed before the call. It takes int ttl and int timeout as parameters. It matches: call(* InetAddress+.isReachable(NetworkInterface, int, int)) && args(*, ttl, timeout).

A violation occurs when a monitored event is triggered under the pointcut conditions described above and its associated violation handler is activated.
