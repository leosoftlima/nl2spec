This specification is written in the EVENT formalism.

This specification is called InetAddress_IsReachable.


The following events are monitored in this specification:

isreachable is observed before the call as a event. It takes int timeout. It matches: call(* InetAddress+.isReachable(int)) && args(timeout).

isreachable is observed before the call as a event. It takes int ttl and int timeout. It matches: call(* InetAddress+.isReachable(NetworkInterface, int, int)) && args(*, ttl, timeout).

A violation occurs when a monitored event is triggered under the pointcut conditions described above and its associated violation handler is activated.

The violation message that must be used in the JSON output is:

"__DEFAULT_MESSAGE"

The generated JSON must include the field:

"domain": "net"
